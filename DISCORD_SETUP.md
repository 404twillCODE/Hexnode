# Nodexity Discord Server Plan

This is a clean, bold, website-inspired Discord setup that works now and is
ready for a full public release later.

## 1) Server Name + Branding
- **Server name:** `Nodexity`
- **Tagline:** `Local-first Minecraft Infrastructure`
- **Visual style:** dark UI, neon-green accents, clean monospace look
- **Icon:** Nodexity logo, simple mark on dark background
- **Banner ideas:** `Resources, not restrictions.` or `Local-first server management`

## 2) Categories + Channels (clean + bold)

**CATEGORY: START HERE**
- `#welcome` (branding intro + quick start)
- `#rules` (short list, clean)
- `#announcements` (read-only)
- `#status` (release state + what’s available)

**CATEGORY: COMMUNITY**
- `#general`
- `#introductions`
- `#showcase` (screenshots, builds, setups)
- `#off-topic`

**CATEGORY: SUPPORT**
- `#help`
- `#known-issues`
- `#install-help` (optional)

**CATEGORY: FEEDBACK**
- `#feature-requests`
- `#bug-reports`
- `#ui-feedback`
- `#polls`

**CATEGORY: DEV / UPDATES**
- `#dev-log`
- `#roadmap`
- `#changelog`

**CATEGORY: MEDIA**
- `#screenshots`
- `#clips`

**CATEGORY: TEAM (private)**
- `#staff`
- `#mod-notes`
- `#release-coordination`

## 3) Roles (release-ready)
- **Owner**
- **Admin**
- **Moderator**
- **Developer**
- **Contributor**
- **Early Access**
- **Community**
- **Bot**

Optional roles:
- `Server Owner`, `Builder`, `Plugin Dev`, `Performance Nerd`

## 4) Permissions (simple + strong)
- `#announcements` and `#rules` are **read-only**
- `#staff` and `#mod-notes` are private
- `#bug-reports` and `#feature-requests` allow messages + threads
- `@everyone` can post in Community + Support + Feedback

## 5) Welcome Message (copy-paste)
```
Welcome to Nodexity.

Nodexity is a local-first Minecraft server management platform.
No cloud lock-in. Everything stays on your machine.

Get started:
• Read #rules
• Say hi in #introductions
• Drop ideas in #feature-requests
• Report issues in #bug-reports

We’re pre-release — follow #announcements for updates.
```

## 6) Rules (short + clean)
```
1) Be respectful
2) No harassment, hate, or spam
3) Keep support in support channels
4) No piracy or illegal content
5) Mods have final say
```

## 7) Release-Ready Extras (recommended)
- **Auto-mod:** block spam, links from new users
- **Bot:** Carl-bot or Dyno (roles + moderation)
- **Welcome DM:** send “What is Nodexity?” + Discord invite + GitHub link
- **Server guide:** pin in `#welcome` (short + bold)

## 8) Build It Now (step-by-step)
1. Create server → name **Nodexity**
2. Set icon + banner (dark + neon green)
3. Create categories + channels (list above)
4. Create roles in order (Owner → Bot → Community)
5. Lock down `#announcements` + `#rules`
6. Post welcome message + rules
7. Add a moderation bot (Dyno or Carl-bot)
8. Test with a second account (permissions + visibility)
