# GitHub repository description (change in Settings)

Update the repository description in GitHub so it matches the Nodexity brand.

## Where to change

1. Open the repo: **https://github.com/404twillCODE/Nodexity**
2. Go to **Settings** (tab) → **General**.
3. In **Repository name**, ensure it is **Nodexity** (if you have already renamed it from Hexnode).
4. In **Description**, set a short line for the repo. Suggested:

   **Suggested description (short):**
   ```
   Nodexity – Local-first Minecraft server manager (desktop app + website). Electron + React + Next.js.
   ```

   Or shorter:
   ```
   Nodexity – Local-first Minecraft server management. Desktop app & website.
   ```

5. Optionally set **Website** to your live site: `https://404twillCODE.github.io/Nodexity/`
6. Save.
